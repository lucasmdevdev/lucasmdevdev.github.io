🔍 Web Scraping Without Getting Blocked

March 2026 - Best practices for ethical web scraping

1. Rotate Your User-Agent

const userAgents = [
  'Mozilla/5.0 (Windows NT 10.0; Win64; x64)...',
  'Mozilla/5.0 (Macintosh; Intel Mac OS X)...',
  // ... more agents
];

const randomAgent = userAgents[Math.floor(Math.random() * userAgents.length)];
const response = await fetch(url, {
  headers: { 'User-Agent': randomAgent }
});

2. Add Random Delays

async function fetchWithDelay(url) {
  await new Promise(r => setTimeout(r, Math.random() * 3000 + 1000));
  return fetch(url);
}

3. Use Residential Proxies

Free: free-proxy-list.com

Paid (recommended): Bright Data, Oxylabs, Smartproxy

4. Use Headless Browsers (Playwright, Puppeteer)

const browser = await chromium.launch();
const page = await browser.newPage();
await page.goto(url, { waitUntil: 'networkidle' });
const data = await page.evaluate(() => document.documentElement.innerHTML);

5. Respect robots.txt & Terms of Service

  • ✅ Read target site's robots.txt
  • ✅ Check their terms of service
  • ✅ Respect rate limits
  • ✅ Add a User-Agent with contact info
