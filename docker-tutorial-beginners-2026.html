Docker Tutorial for Beginners

What is Docker?

Docker containers let you package your app + all dependencies. Run anywhere.

Install Docker

Download from docker.com

Create Dockerfile

FROM node:20
WORKDIR /app
COPY . .
RUN npm install
CMD ["npm", "start"]

Build

docker build -t my-app .
docker run -p 3000:3000 my-app
