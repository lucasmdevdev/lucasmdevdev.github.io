Votre première automatisation avec Make en 2026 : guide pas à pas

Vous avez entendu parler de l'automatisation no-code mais vous n'avez jamais sauté le pas ? Ce guide vous accompagne étape par étape — en moins de 30 minutes, sans une seule ligne de code.

Ce qu'on va construire

Objectif : Chaque fois que vous recevez un email contenant un lien, il est automatiquement sauvegardé dans une base Notion "À lire plus tard".

C'est simple, pratique, et parfait pour comprendre la logique de Make.

Prérequis

  • Un compte Make gratuit (1000 opérations/mois offertes)
  • Un compte Gmail ou Outlook
  • Un compte Notion (gratuit)
  • ~20-30 minutes

Étape 2 — Créer la base Notion "À lire"

  1. Dans Notion, créez une nouvelle page
  2. Ajoutez un database (type "Table")
  3. Colonnes à créer :
    • Titre (text) — le sujet de l'email
    • Lien (URL) — le premier lien trouvé
    • Date (date) — quand l'email a été reçu
    • Source (text) — l'expéditeur

Étape 3 — Créer le scénario Make

3.1 — Nouveau scénario

Dans Make, cliquez sur "Create a new scenario". Vous arrivez sur l'éditeur visuel — c'est là que la magie opère.

3.2 — Le trigger : Gmail

Cliquez sur le "+" central → cherchez "Gmail" → choisissez "Watch Emails".

Configurez :

  • Folder : INBOX
  • Criteria : All mail
  • Maximum number of emails : 5 (pour débuter)

Make va vous demander de connecter votre compte Gmail via OAuth — c'est sécurisé.

3.3 — Le filtre : "contient un lien ?"

Ajoutez un filtre entre Gmail et Notion :

  • Condition : Body → Contains → http

Ça évite de sauvegarder tous vos emails, seulement ceux avec des liens.

3.4 — L'action : Notion

Ajoutez un module Notion → "Create a Database Item". Configurez les champs :

  • Titre → {{Subject}}
  • Date → {{Date}}
  • Source → {{From: Name}}

3.5 — Activer le scénario

Cliquez sur "Run once" pour tester. Si tout fonctionne, activez le scheduling (toutes les 15 minutes).

Résultat

Désormais, chaque email avec un lien apparaît automatiquement dans votre Notion "À lire plus tard". Plus jamais un article oublié au fond de votre boîte mail.

Pour aller plus loin

  • Version avancée : Ajouter un module Claude/GPT pour résumer automatiquement l'article
  • Version pro : Filtrer sur un label Gmail, ajouter une priorité basée sur l'expéditeur
  • Version équipe : Notification Slack quand un article important arrive
